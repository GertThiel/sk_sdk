= SalesKing SDK

A bunch of classes to get a quick start creating SalesKing App's, using oAuth and
the API.

Still in a very early stage

== Install

    gem install sk_sdk

Dependencies (gem's):

* activeresupport
* curb

== Usage

The classes must be explicitly required

  require 'sk_sdk/sign_req'
  require 'sk_sdk/oauth'
